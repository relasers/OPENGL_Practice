#구현내용
폴리곤영역의 클리핑을 제외한 과제로 주어진 모든 부분의 구현이 완료되었습니다,
#추가내용
내려오는 도형을 알맞게 매칭시키면 그 도형이 색칠됩니다.
	->위치판정은 메인 도형보다 넓게 설정되어 있습니다.
	->대신 도형의 모양과 크기는 정확하게 맟춰야합니다.

#완벽하게 구현하지 못한내용
다각형 클리핑 알고리즘 :: # 대신 서더랜드-핫지먼 알고리즘을 통해 기초적인 사각형 영역의 클리핑을 구현하였습니다.

====================<조작 도형>=================================
a,d 	: 좌우 이동을 합니다
w,s	: 확대 축소가 이루어집니다.
q	: 좌우 반사가 이루어집니다
e	: 상하 반사가 이루어집니다. 
	:: 메인 도형을 회전시키면 확인 가능합니다.
r	: 꾹 눌러 도형을 천천히 회전시킵니다.
j,i,o,p 	: 메인도형이 각각 삼각형,사각형,오각형,육각형으로 변환됩니다.

x	: 임의의 도형을 생성시킵니다.
====================<구현내용 및 진  행>==================================
* 점수 획득 조건 
	:: 메인 도형과 내려오는 도형과 겹쳤을 때
	:: 메인 도형과 정점 갯수가 같아야 합니다
	:: 메인 도형과 크기가 같아야합니다. ::회전과, 반사, 밀림까지 일치할 필요는 없습니다.
	위 조건을 모두 만족하면 점수를 획득할 수 있습니다.

* 주기적으로 상단에서 도형이 내려옵니다.
	-> 도형은 무작위의 크기를 가집니다.
	-> 도형은 무작위의 각도로 회전해있습니다.
	-> 도형은 일정 확률로 밀림 또는 반사가 적용되어 있습니다.
* 일정 시간마다 도형이 내려오는 속도가 변합니다.

* 가끔 테두리 색깔이 다른 도형이 내려오는데 이 경우 메인 도형과 정점갯수와 크기가 달라도
  접촉만 하면 점수를 획득할 수 있습니다.

* 점수 도형을 가득 채운상태에서 한번 더 점수를 획득하면 게임이 리셋됩니다 -> [리-셋].
